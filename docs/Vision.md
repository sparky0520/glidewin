# Project: GlideWin

## Goal

Build a Windows-native AI desktop companion inspired by Glide.

### MVP Features

* Global hotkey
* Push-to-talk
* Screenshot capture
* Speech-to-text
* Send screenshot + transcript to GPT-4o/Gemini
* Display answer
* Optional text-to-speech

### Explicitly Out of Scope

* Accounts
* Billing
* Cloud backend
* OAuth
* Notion/Gmail integrations
* Multi-user support
* Agents
* Cursor pointing
* Long-running automation
